## [See2gether](http://ooppsss60.github.io/see2gether)
  Serverless app for synchronous viewing video together.
  
  How to use:
  1. Allow video autoplay.
  2. Share link with your friend.
  3. Paste video url.
  4. Watch!
  
  Supported video format: **.mp4 .ogg .ogv .webm**
